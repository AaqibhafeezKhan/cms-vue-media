# cms-vue-media

Vue 2 + TypeScript microfrontend for digital asset management in the CMS Ultra platform.

## Features

- 🖼️ Media grid with tab filtering (All / Images / Videos / Docs)
- 📊 Asset stats dashboard
- 🔍 Hover overlay with quick view/copy actions
- ⬆️ Upload interface

## Live Demo

🌐 **Standalone:** https://AaqibhafeezKhan.github.io/cms-vue-media  
🌐 **In Shell:** https://AaqibhafeezKhan.github.io/cms-root-orchestration/media

## Local Development

```bash
npm install
npm start   # → http://localhost:8082
```

## Build

```bash
npm run build   # outputs dist/main.js as SystemJS module
```
